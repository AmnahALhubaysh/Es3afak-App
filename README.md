# Es3afak - إسعافك 🚑

### 📝 نبذة عن المشروع (Project Overview)
تطبيق **إسعافك** هو تطبيق جوال تفاعلي مخصص لتعليم مهارات الإسعافات الأولية باللغة العربية والانجليزية. يهدف المشروع إلى تقديم تدريب شامل وسهل الوصول لمساعدة الناس على الاستجابة بشكل صحيح في حالات الطوارئ.

**Es3afak** is an interactive mobile application designed to teach first aid skills in Arabic and Engilsh. The goal is to provide inclusive and accessible training to help people respond correctly in emergencies.

---

### ✨ المميزات الرئيسية (Key Features)
*   **دروس تفاعلية:** خطوات عملية لكل حالة طارئة.
*   **دعم كامل للغة العربية:** واجهات ومحتوى مخصص للمجتمع العربي.
*   **قاعدة بيانات محلية:** استخدام **SQLite** لضمان عمل التطبيق بدون إنترنت.
*   **واجهة مستخدم سهلة:** تصميم بسيط يركز على سرعة الوصول للمعلومة.

---

### 🛠 التقنيات المستخدمة (Tech Stack)
*   **Framework:** Flutter 📱
*   **Language:** Dart 🎯
*   **Database:** SQLite 🗄️
*   **Tools:** Git & GitHub
