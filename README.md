Es3afak - إسعافك 🚑

A comprehensive, interactive mobile application designed to provide essential first aid training and emergency response guidance.

🌟 Key Features
- **Bilingual Support:** The application fully supports both **Arabic** and **English** languages to ensure accessibility for a wider range of users.
- **Interactive Learning:** Step-by-step interactive guides for handling various medical emergencies.
- **Offline Access:** Powered by **SQLite** to ensure all life-saving information is available anytime, even without an internet connection.
- **User-Centric Design:** A clean and intuitive UI built with Flutter for quick navigation during emergencies.

 🛠 Tech Stack
- **Framework:** [Flutter](https://flutter.dev) 📱
- **Language:** [Dart](https://dart.dev) 🎯
- **Database:** [SQLite](https://www.sqlite.org) 🗄️
- **Version Control:** Git & GitHub

🎓 Graduation Project
This project is developed as the final graduation project for a Bachelor’s degree in **Computer Science** (Class of 2026). It demonstrates the integration of mobile development with local database management to solve real-world problems.
